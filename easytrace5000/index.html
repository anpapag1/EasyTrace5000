Accessibility Navigation

Focus Zones

The interface is divided into zones. Press F6 to cycle focus between them:

  1. Toolbar: Top actions and view modes
  2. Operations Tree: Left sidebar (File navigation)
  3. Properties: Right sidebar (Settings)

Note: Use the Skip Link (Tab on load) to jump to Canvas.

Global Shortcuts

F6 Cycle focus zones
Delete Remove selected operation
Esc Deselect / Close modal
V Toggle layer visibility
F1 Show this help

Quick Tips

  • Drag files directly onto the canvas to auto-detect operation type
  • Use the eye icon in the tree to toggle layer visibility
  • Mouse wheel zooms; middle-click or right-click drags to pan
  • Check the Visualization panel (bottom-left) for debug options
